# Просматривает посты для указанных телеграм каналов, фильрует за определенный период и по популярности (просмотры, реакции, комментарии), отправляет посты в указанный тг канал изменяя текст.



Немного информации, так же много информации о настройках в config ini в комментариях.


Если нужно очистить историю в файле sent_posts то вставить просто вот это:
{
  "last_updated": "2026-01-12T18:38:51.040897",
  "sent_posts": [
  ]
}
Ну тобишь очистить список sent_posts.


Для старта нужно Добавить api_id и api_hash, я убрал свои.

Апи ключ нейросети можно вставить свой, но сейчас там мой стоит, можно и не менять пока квота не израсходована.

Ну и указать с каких каналов брать посты в виде их айди, и в какой канал пересылать, указывать тег канала, с приватным не пробовал, если че можно в избранное, для этого нужно указать айди свой,
переслать любое сообщение из избранного в бота @JsonDumpBot и в ответе найти ключ chat и взять значение в виде айди.

Описание настроек в config.ini.

если не пересылаются посты то тоже перезапустить скрипт попробовать.
